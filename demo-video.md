# Demo Video

## Concepto

Top movers diario estilo financial-news.

Tema:

```text
Las acciones que mas subieron y bajaron hoy.
```

## Duracion

45-60 segundos.

## Formato

- Target demo: 9:16 para Instagram/Reels/Shorts.
- Variante futura: 16:9 para YouTube.

## Estructura

### 1. Hook

```text
Estas fueron las acciones que mas se movieron hoy.
```

Visual:

- Titulo grande.
- Fondo market dashboard.
- Tickers en movimiento.

### 2. Top subidas

5 escenas rapidas.

Cada accion:

- Ticker.
- Nombre empresa.
- % cambio.
- Mini chart 1D o 5D.
- Catalizador corto.

Ejemplo:

```text
NVDA +4.8%
Catalizador: demanda de chips IA y momentum del sector.
```

### 3. Top bajadas

5 escenas rapidas.

Cada accion:

- Ticker.
- Nombre empresa.
- % cambio.
- Mini chart 1D o 5D.
- Catalizador corto.

Ejemplo:

```text
TSLA -3.2%
Catalizador: presion por margenes y dudas sobre entregas.
```

### 4. Cierre

```text
Para manana, vigila resultados, guidance y datos macro.
```

Visual:

- 3 cards: earnings, macro, sector momentum.

## Estilo visual

- Inspiracion: Bloomberg / Instagram financiero.
- No copiar marca ni layout exacto.
- Fondo oscuro.
- Numeros grandes.
- Verde para subidas.
- Rojo para bajadas.
- Graficas animadas.
- Captions cortos.
- Ritmo rapido.
